# Legal Document Translator - Color Palette

Color palette inspired by DeepL's minimal, professional aesthetic.

## Primary Colors

### Accent Color
- **DeepL Teal-Cyan:** `#00B4D8`
  - Primary action buttons
  - Links and interactive elements
  - Progress bars
  - Focus states
  - Icon colors

- **DeepL Teal-Cyan (Hover):** `#0099C0`
  - Button hover states
  - Interactive element hover states

### Text Colors
- **Primary Text:** `#000000` (Pure black)
  - Headings
  - Body text
  - Labels

- **Secondary Text:** `#666666` (Medium gray)
  - Subtitles
  - Helper text
  - Muted content

## Background Colors

### Main Backgrounds
- **Page Background:** `#fafafa` (Very light gray)
  - Main application background
  - Creates subtle contrast with white cards

- **Card Background:** `#ffffff` (Pure white)
  - Main content cards
  - Form containers
  - Text content areas

### Section Backgrounds
- **Light Section:** `#f8f9fa` (Light gray)
  - Stats sections
  - Info boxes
  - Alternating sections

- **Medium Section:** `#f5f5f5` (Medium-light gray)
  - Stat cards
  - Column headers
  - Code blocks
  - Table headers

- **Subtle Background:** `#f0f0f0` (Subtle gray)
  - Progress bar backgrounds
  - Inline code backgrounds

## Border Colors

- **Primary Border:** `#e5e5e5` (Light gray)
  - Card borders
  - Input borders
  - Section dividers
  - Table borders

## Design Principles

1. **Minimal & Clean:** Use subtle backgrounds to create hierarchy without heavy visual elements
2. **High Contrast:** Pure black text on white/light backgrounds for readability
3. **Subtle Depth:** Light shadows and borders instead of heavy gradients
4. **Consistent Spacing:** Generous white space for clean, professional appearance
5. **Teal Accent:** Use DeepL teal-cyan sparingly for emphasis and actions

## Usage Guidelines

- **White backgrounds** (`#ffffff`) for primary content areas
- **Very light gray** (`#fafafa`) for page background
- **Light gray sections** (`#f8f9fa`, `#f5f5f5`) for visual separation
- **Teal accent** (`#00B4D8`) only for interactive elements and highlights
- **Pure black** (`#000000`) for all text to maintain DeepL's bold typography feel
