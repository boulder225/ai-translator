## Color Palette

- **Primary Text:** `#000000` (Pure black)
- **Icon Color:** `#00B4D8` (DeepL teal-cyan)
- **Background:** `#ffffff` (White) or transparent

## Typography

- **Font:** Inter (700 weight)
- **Fallback:** -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto
- **Style:** Bold, condensed, geometric sans-serif
- **Letter Spacing:** -0.02em (slightly tighter)

## Usage Guidelines

1. **Minimum Size:** Maintain readability at 120px width minimum
2. **Spacing:** Provide clear space around logo (at least 20% of logo height)
3. **Background:** Works on white or transparent backgrounds
4. **Scaling:** All SVGs are scalable - maintain aspect ratio
5. **Do Not:**
   - Add gradients, shadows, or textures
   - Modify colors
   - Stretch or distort
   - Use on busy backgrounds

## Technical Notes

- All logos are SVG format for scalability
- Optimized for web and print use
- Font loading via Google Fonts (Inter)
- Compatible with all modern browsers
